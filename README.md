<br>

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/descreetStudios/Wink.git
cd Wink
```
### 2. Configure the project (CMake)
```bash
cmake -S . -B build
```
### 3. Build the engine
```bash
cmake --build build
```
### 4. Run
```bash
./out/build/Sandbox
```
